Starlight Programming Language

Starlight is a modern interpreted programming language designed for simplicity, power, and extensibility.

It features a clean syntax, first-class async support, modular packages, and a flexible runtime designed for scripting, tooling, and automation.

Use the sidebar to explore the language, its packages, CLI tools, and full language specification.
