```
Starlight Tutorials

1. Variables

In Starlight, you must declare variables using let or define.

let x = 10
```

let y = 20
define pi = 3.14
```

2. Loops

Use for or while loops to iterate.

let arr = range(1, 6)
```

let sum = 0

for x in arr {
sum = sum + x
}
sldeploy sum  # Outputs: 15
```

3. Conditionals

let n = 7
```

if n % 2 == 0 then
sldeploy "Even"
else
sldeploy "Odd"
```

4. Functions & Arrow Functions

define add = (a, b) => a + b
```

sldeploy add(3, 5)  # Outputs: 8

let square = x => x * x
sldeploy square(4)   # Outputs: 16
```

5. Arrays

let arr = [1, 2, 3, 4, 5]
```

let doubled = map(arr, x => x * 2)
sldeploy doubled  # Outputs: [2, 4, 6, 8, 10]
```

6. Objects

let person = define {
name: "Alice",
age: 20,
greet: x => "Hello " + x
```

}

sldeploy person.name         # Alice
sldeploy person.greet("Bob") # Hello Bob
person.city = "NYC"
sldeploy person  # { name: Alice, age: 20, greet: , city: NYC }
```

7. Built-in Functions

len([1,2,3])        # 3
```

len("hello")       # 5
keys({a:1, b:2})    # ["a", "b"]
values({a:1, b:2})  # [1, 2]

sleep(1000)         # Waits 1 second
ask("Name?")      # Prompts user input
```

8. Higher-order functions

let nums = range(1, 6)
```

let evens = filter(nums, x => x % 2 == 0)
sldeploy evens   # [2,4]

let sum = reduce(nums, (acc, x) => acc + x, 0)
sldeploy sum     # 15
